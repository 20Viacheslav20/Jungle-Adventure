Jungle Adventure is an exciting 2D game developed using the Unity Framework. Explore the exciting jungle world and face various challenges on your way to victory.

---

### Game control

+ **A / Num 4** — move left 
+ **D / Num 6** — move right
+ **W / Space / Num 8** — move up
+ **R** — restart lvl
+ **Esc** — pause game
